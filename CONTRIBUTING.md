# Contributing to StoryForge

Thank you for your interest in contributing! We welcome bug reports, feature ideas, documentation improvements, and code contributions.

## How to Contribute

### Report Bugs
- Check existing issues first.
- Open a new issue with:
  - Clear description
  - Steps to reproduce
  - Expected behavior

### Propose Features
- Open an issue labeled “Feature Request.”
- Explain what the feature does and why it’s useful.
- Discussion is encouraged before major changes.

### Submit Pull Requests
1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Ensure tests pass.
5. Submit a clear pull request and link related issues.
